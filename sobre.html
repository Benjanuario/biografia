Minha Jornada

De professor a criador de soluções digitais. Cada passo foi uma lição, cada desafio uma oportunidade de crescimento.

1999

Nasce um Moçambicano

Vim ao mundo em Caia, província de Sofala, com a energia do rio Zambeze e a riqueza cultural moçambicana como herança.

2020

O Chamado para Ensinar

Iniciei minha jornada como professor na Província da Zambézia, descobrindo minha paixão por compartilhar conhecimento e impactar vidas.

2022

Primeiro Encontro com o Código

Movido pela necessidade de melhorar minhas aulas, dei meus primeiros passos no mundo da programação. HTML, CSS e JavaScript se tornaram meus novos alfabetos.

2024

Nascimento do Stunner System

Fundei o Stunner System como incubadora dos meus projectos. Aqui, cada ideia encontra o ambiente perfeito para se tornar realidade.

Descobri o poder das IAs generativas (ChatGPT, DeepSeek, Claude, Qwen, etc) como mentores 24/7, acelerando meu aprendizado de forma exponencial.

2025

Primeiro Grande Projecto: Gerador de Planos de Aula e Quinzenal

Desenvolvi meu primeiro aplicativo completo - uma ferramenta para professores gerarem planos de aula e quinzenais. Aprendi na prática o significado de "aprender fazendo".

2026

Soluções para Problemas Reais

Lancei o Actualizações CFM resolvendo um problema de comunicação dos passageiros, e o MozaIA para democratizar o conhecimento sobre inteligência artificial em Moçambique.

Futuro

Próximos Horizontes

Domínio de Python, React, e machine learning. Expansão do Stunner System para incubar projectos de outros desenvolvedores moçambicanos. Contribuir para o ecossistema tech de Moçambique.

Minha Filosofia de Desenvolvimento

Soluções com Alma

Cada projecto nasce de uma dor real, uma necessidade específica. Não crio apenas código - crio experiências que melhoram vidas.

Aprendizado Contínuo

Vejo cada linha de código como uma oportunidade de aprendizado. As IAs são meus mentores, os bugs são meus professores, e cada projecto é uma nova escola.

Tecnologia que Serve

Acredito que a tecnologia deve ser acessível e útil para todos. Meu objectivo é criar ferramentas que empoderem pessoas, especialmente na realidade moçambicana.

"O maior código que podemos escrever não está nos computadores, mas na transformação positiva que causamos na vida das pessoas."
— Benjamim Januário Paulino

Esta jornada é apenas o começo. O melhor ainda está por vir.
